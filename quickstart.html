üöÄ Quick Start

Option 1: View Locally

  • Open index.html in your web browser
  • The dashboard will run with simulated data

Option 2: Deploy to ESP32

  1. Upload all files to your ESP32 web server
  2. Configure your ESP32's WiFi settings
  3. Access via ESP32's IP address
  4. Update script.js to fetch real sensor data
